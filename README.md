# MT5 Quant Framework (Demo)

This repository contains a modular MetaTrader 5 (MT5) trading framework 
designed to demonstrate quant-style architecture for:

- Market structure analysis
- Fair Value Gap (FVG) detection
- Risk-based position sizing
- Time & spread filtering
- Modular execution logic

⚠️ This is a demonstration framework.  
Entry logic is simplified and does NOT represent a production trading edge.

## Features
- Swing-based market structure bias
- FVG mitigation detection
- Fixed fractional risk management
- Daily drawdown protection
- Modular, extensible architecture

## Architecture
Core components are separated into reusable modules:

- Market Structure Engine  
- FVG Detection Module  
- Risk Management Engine  
- Execution Engine  

## Use Cases
- Strategy prototyping
- Execution framework development
- Risk management testing
- Quant-style system architecture examples

## Disclaimer
This code is for educational and framework demonstration purposes only.  
It is not intended for live trading or financial advice.
